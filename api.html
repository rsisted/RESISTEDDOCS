FIRST OF ALL

RSBlocks use a different wiring method and will use raycasting for it's wiring. This is because we are unable to create Attachments (inputs and outputs) for the blocks.


SineVisualizer Block

This block takes soundwaves from nearby sound sources and visualizes it. You can see the visualization in the following method:

1. Grab a valuebrick and place it infront of the studded surface. Not smooth, not welded, and it should have atleast a .1 gap inbetween.



2. Place a sound source nearby

Done!


You can always add extra stuff to your likings (such as a Display). The Sinewave ASCII is saved as "SineSignal"
